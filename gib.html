About

This is a website that utilizes an open GIB rest API that is also used by the Bridge Base Online arcade gamemode. Currently it can simulate hand bidding situations, so GIB’s bidding behavior can be studied more easily which helps us improve it. This website will eventually be able to simulate how GIB would play a hand of cards given the bids ahead of time, though I find this less useful and it is much more complicated to implement. GIB also has APIs for providing bid meanings and dealing cards, which I may add support for eventually.
